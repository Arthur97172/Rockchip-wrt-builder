## 🤔 项目介绍 [![](https://img.shields.io/badge/-项目介绍-FFFFFF.svg)](#-项目介绍-)
**目标是提供一个纯净的ImmortalWrt系统，支持24.10.x和25.12.x版本编译，并可选择是否包含Docker。**
***编译快速，基本上5分钟就可以完成编译工作。***

> [!TIP]
> 😂此固件为 **非官方构建，不保证完全无BUG** ，请知悉😂
